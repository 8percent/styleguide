Eight Style Guides
===========

## Purpose

8퍼센트를 만들어가는 사람들이 지향하는 코드 스타일을 정의한 문서입니다. 스타일 가이드를 활용하여 코드의 가독성을 높이고 다같이 일을 잘 할 수있도록 하는것이 목표입니다.

## Notice

이 스타일 가이드를 구성하는 항목들은 팀원들의 합의에 의해 지속적으로 개선되거나 변경될 수 있습니다.
